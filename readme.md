# Cashier

A banking service that allows customers to repay what they owe to the creditor bank.

```bash
chmod +x ./run.sh
sudo ./run

# OR
php -S localhost:8000
sudo systemctl start mysql
# If some error occurs
mysql -u root

``sql
CREATE USER 'newuser'@'localhost' IDENTIFIED BY 'some_strong_password';
GRANT ALL PRIVILEGES ON *.* TO 'newuser'@'localhost';
FLUSH PRIVILEGES;
``

composer dump-autoload # --help

```
